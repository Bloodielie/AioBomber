# AioBomber

### Installation

AioBomber requires [Python 3.7+](https://www.python.org/) to run.

Complete installation guide.

```sh
$ git clone https://github.com/Bloodielie/AioBomber.git
$ cd AioBomber
$ pip install -r requirements.txt
```

Install the dependencies.

```sh
$ pip install -r requirements.txt
```

### Start

Start attack

```sh
$ python bomber.py --phone=<your phone> -- cycles=<number of cycles>
```
